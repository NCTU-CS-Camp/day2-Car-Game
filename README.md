# day2-Car-Game

用 pygame 做的賽車小遊戲：開車繞著賽道跑，順時針繞完一圈就加分，撞到賽道邊界會重置。

## 執行方式

先同步專案依賴（第一次執行時會自動建立 `.venv`）：

```bash
uv sync
```

啟動遊戲：

```bash
uv run python main.py
```

## 操作方式

| 按鍵 | 動作 |
|---|---|
| W | 加速 |
| S | 倒車/煞車 |
| J | 左轉 |
| K | 右轉 |
| 滑鼠左鍵 | 點擊右上角 1/2/3 檔位按鈕切換最高速度 |

## 遊戲規則

- 賽道左側「START」是起點/終點線，車子在終點線附近重生。
- 賽道上有 3 個依序排列的檢查點，車子必須**順時針**依序踩過(CP1 → CP2 → CP3)、再回到終點線，才算完成一圈、分數 +1。逆時針繞圈會用錯誤的順序踩到檢查點，不會被記分。
- 撞到賽道邊界(賽道外的草地/裝飾物)會讓分數歸零，車子重生回起點。
- `Best` 會記錄這次執行期間出現過的最高分，重新執行程式就會歸零(不會存檔)。

## 檔案結構

```text
main.py                       進入點，把所有東西串在一起(讀圖、建立物件、主迴圈)
car.py                        Car 物件：油門/轉向/碰撞偵測等物理引擎(含 move()/rotation() 數學輔助函式)
controls.py                   處理鍵盤輸入：W/S 加速倒車、J/K 轉向
score.py                      比賽規則與計分：終點線、檢查點、分數、最高分
speed_and_boundary.py         變速按鈕 UI 與撞牆處理
assets/                       車子與賽道圖片(car.png、track_front.png、track_back.png)
checkpoint_map.png            檢查點位置示意圖

```
